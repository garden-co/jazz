# Specs Subsumption Audit

This audit classifies every file present under `specs/` before the 2026-07-19
subsumption pass. `MERGE-INTO-CRATE-SPEC` files were folded into the owning crate
SPEC chapter and deleted. `OBSOLETE` files were deleted because the crate SPECs
already supersede the old alpha behavior with no remaining content to extract.
`JAZZ-PRIVATE CANDIDATE` files were copied to `/tmp/jazz-private-staging/`
preserving relative paths and intentionally left in `specs/` for Anselm's
separate approval. `KEEP-AT-TOP` files remain at the top level.
