Build a local-first to-do app with Jazz and Svelte. For the full set of options, SvelteKit
setup, and the other frameworks, see [Install](/docs/install/client).

## Create a project

```bash title="Terminal"
pnpm create vite my-jazz-app --template svelte-ts
cd my-jazz-app
pnpm install
```

<Callout type="info">
  This quickstart uses Svelte 5 features (`$state`, `$props`, `$derived`, snippets). Make sure
  you're on Svelte 5 or later.
</Callout>

## Install

```bash title="Terminal"
pnpm add jazz-tools@alpha
```

## Get an app ID

Your app ID namespaces your data for storage and sync. Generate one on
[Jazz Cloud](https://v2.dashboard.jazz.tools) (sync URL: `https://v2.sync.jazz.tools/`).
