## Local-first auth

`LocalFirstAuth` from `jazz-tools/svelte` exposes `secret`, `isLoading`, and `login` / `signOut`
to switch or clear accounts. See [Local-first auth](/docs/auth/local-first-auth) for the full
flows, including recovery passphrase and passkey backup.
