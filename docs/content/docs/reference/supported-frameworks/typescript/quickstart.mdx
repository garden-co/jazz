Build a local-first to-do app with Jazz using the low-level client API&hairsp;—&hairsp;no
framework bindings. For the full set of options and the framework adapters, see
[Install](/docs/install/client).

## Create a project

```bash title="Terminal"
mkdir my-jazz-app && cd my-jazz-app
pnpm init
pnpm add vite typescript
```

## Install

```bash title="Terminal"
pnpm add jazz-tools@alpha
```

## Get an app ID

Your app ID namespaces your data for storage and sync. Generate one on
[Jazz Cloud](https://v2.dashboard.jazz.tools) (sync URL: `https://v2.sync.jazz.tools/`).

## List to-dos

Subscribe to a query to get real-time updates as data changes, regardless of where the change
originates.

<include cwd lang="ts" meta='title="src/main.ts"'>
  ../examples/docs/todo-client-localfirst-ts/src/quickstart.ts#list-ts
</include>

The callback receives a `{ all, delta }` object&hairsp;—&hairsp;`all` is the current full result
set, and `delta` is an array of row-level changes (each with a `kind`: added, removed, or
updated).
