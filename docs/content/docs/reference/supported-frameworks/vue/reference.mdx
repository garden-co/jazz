## Local-first auth

`useLocalFirstAuth()` from `jazz-tools/vue` loads the device secret (or generates one on first
run) and exposes `login` / `signOut` to switch or clear accounts. See
[Local-first auth](/docs/auth/local-first-auth) for the full flows, including recovery passphrase
and passkey backup.
