Build a local-first to-do app with Jazz and React. For the full set of options, native/Expo
setup, and the other frameworks, see [Install](/docs/install/client).

## Create a project

```bash title="Terminal"
pnpm create vite my-jazz-app --template react-ts
cd my-jazz-app
pnpm install
```

## Install

```bash title="Terminal"
pnpm add jazz-tools@alpha
```

## Get an app ID

Your app ID namespaces your data for storage and sync. Generate one on
[Jazz Cloud](https://v2.dashboard.jazz.tools) (sync URL: `https://v2.sync.jazz.tools/`).
