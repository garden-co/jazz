## Query subscriptions

Subscribe to query results reactively. React re-renders with a fresh array on each
change&hairsp;—&hairsp;it uses React's own diffing rather than the in-place reconciliation the
Vue, Svelte, and Solid bindings use. See [Reading Data](/docs/reading/queries) for loading states
and Suspense.

## Local-first auth

`useLocalFirstAuth()` from `jazz-tools/react` returns `{ secret, isLoading, login, signOut }`: it
loads the device secret (or generates one on first run), and `login` / `signOut` switch or clear
accounts. See [Local-first auth](/docs/auth/local-first-auth) for the full flows, including
recovery passphrase and passkey backup.
