## 1. Hosted Database Server (From Us)

Current onboarding is manual.

- App IDs are provisioned manually.
- Request provisioning in the `jazz2-adopters` Discord channel.
- Include:
  - app name
  - JWKS endpoint

## 2. Run a Database Server Yourself (CLI)

Self-hosting flow is documented from the runnable examples and CLI workflows.
This page will expand with concrete production/dev presets.
