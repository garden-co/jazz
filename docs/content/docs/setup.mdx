## 1. Hosted Database Server (From Us)

Current onboarding is manual.

- App IDs are provisioned manually.
- Request provisioning in the `jazz2-adopters` Discord channel.
- Include:
  - app name
  - JWKS endpoint

### Backend Identity Pattern

- Client-side usage enforces permissions for the local user context.
- Backend usage must extract requester JWT and create a user-scoped client/session.
- Read/write and permissions sections reference this model.
