## Deleting Data

Use the `delete` method:

<include>../../../examples/react/src/concepts/mutations.tsx#delete-task</include>

Deletes are "soft" by default - the row is marked as deleted but its history is preserved. This allows undo and sync consistency.

## Instant Updates

All mutations apply instantly to local state:

<include>../../../examples/react/src/concepts/mutations.tsx#instant-updates</include>

The sync server receives changes in the background. If you're offline, changes queue locally and sync when you reconnect.
