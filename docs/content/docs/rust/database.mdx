### Supported Column Types

| Type | Description |
|------|-------------|
| `BOOL` | Boolean value |
| `I32` | 32-bit signed integer |
| `U32` | 32-bit unsigned integer |
| `I64` | 64-bit signed integer |
| `F64` | 64-bit float |
| `STRING` | UTF-8 text |
| `BYTES` | Binary data |
| `BLOB` | Large binary data |
| `REFERENCES TableName` | Foreign key |
