Jazz uses a WASM-compiled Rust database that runs in the browser. Here's how to set it up.

## Initializing the Database

```typescript title="database.ts"
import init, { WasmDatabase } from 'groove-wasm';

export async function initDatabase() {
  // Initialize the WASM module
  await init();

  // Create an in-memory database
  const db = new WasmDatabase();

  // Or create a persistent IndexedDB-backed database
  // const db = await WasmDatabase.withIndexedDb('my-app');

  // Initialize your schema
  db.init_schema(schemaSQL);

  return db;
}
```

## Schema Initialization

Pass your SQL schema to `init_schema`:

```typescript title="schema.sql"
CREATE TABLE Users (
  id STRING NOT NULL,
  name STRING NOT NULL,
  email STRING NOT NULL
);

CREATE TABLE Tasks (
  id STRING NOT NULL,
  title STRING NOT NULL,
  completed BOOL NOT NULL,
  user REFERENCES Users NOT NULL
);
```

```typescript
import schema from './schema.sql?raw';  // Vite raw import

db.init_schema(schema);
```

## Database Methods

### init_schema

Initialize the database schema:

```typescript
db.init_schema(schemaSQL);
```

### list_tables

Get all table names:

```typescript
const tables = db.list_tables();
console.log('Tables:', tables);  // ['Users', 'Tasks']
```

### execute

Execute a SQL statement (INSERT, UPDATE, DELETE):

```typescript
db.execute(
  'INSERT INTO tasks (id, title, completed) VALUES (?, ?, ?)',
  ['task-1', 'New task', false]
);
```

### query

Execute a SELECT query:

```typescript
const tasks = db.query('SELECT * FROM tasks WHERE completed = ?', [false]);
```

## Persistence Options

### In-Memory (Default)

Data is lost when the page reloads:

```typescript
const db = new WasmDatabase();
```

### IndexedDB Persistence

Data persists across sessions:

```typescript
const db = await WasmDatabase.withIndexedDb('my-app-name');
```

### Check for Existing Database

```typescript
const exists = await WasmDatabase.hasPersistedDatabase('my-app-name');
if (exists) {
  console.log('Loading existing database...');
}
```

### Delete Persisted Database

```typescript
await WasmDatabase.deletePersistedDatabase('my-app-name');
```

## Using with React

Wrap your app with `JazzProvider`:

```tsx title="App.tsx"
import { JazzProvider } from '@jazz/react';
import { app } from './generated/client';

function App() {
  const [db, setDb] = useState(null);

  useEffect(() => {
    initDatabase().then(setDb);
  }, []);

  if (!db) return <div>Loading...</div>;

  return (
    <JazzProvider database={db}>
      <YourApp />
    </JazzProvider>
  );
}
```

Then use hooks to interact with data:

```tsx
import { useAll, useJazz } from '@jazz/react';
import { app } from './generated/client';

function TaskList() {
  const [tasks, loading] = useAll(app.tasks);
  const db = useJazz();

  const createTask = () => {
    app.tasks.create(db, { title: 'New', completed: false });
  };

  // ...
}
```
