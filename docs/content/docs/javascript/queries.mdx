## Supported Query Features

The current implementation supports:

- Filtering with equality conditions (AND logic)
- Loading related data through foreign key references
- Real-time subscriptions with automatic updates
- LIMIT and OFFSET for pagination
- Full TypeScript type inference from schema

### Not Yet Implemented

The following SQL features are planned but not yet available:

- ORDER BY (results currently ordered by ObjectId/insertion time)
- GROUP BY and HAVING
- Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
- LEFT/RIGHT/OUTER JOINs (only INNER JOIN)
- Complex subqueries
- Raw SQL queries from JavaScript
