The `@jazz/client` package provides the core JavaScript API for Jazz, usable in any JavaScript environment (browser, Node.js, etc.).

## Installation

```bash
pnpm add @jazz/client @jazz/schema
```

## Core Concepts

The JavaScript API provides:

- **Generated client**: Type-safe table descriptors from your SQL schema
- **Query builder**: Fluent API with `where()` and `with()` methods
- **Live subscriptions**: Callbacks that fire when data changes
- **Mutations**: Type-safe `create()`, `update()`, `delete()` methods
