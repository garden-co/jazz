Jazz provides type-safe mutations through the generated client API.

## Using the Generated Client

```typescript title="mutations/using-client.ts"
import { app } from '../generated/client';
import { createDatabase } from '../generated/client';

// Get a bound database instance
const db = createDatabase(wasmDb);

// Or use the app directly with db parameter
// Create a new task
const taskId = app.tasks.create(db, {
  title: 'Complete project documentation',
  description: 'Write comprehensive docs for all modules',
  completed: false,
});

console.log('Created task with ID:', taskId);

// Update an existing task
app.tasks.update(db, taskId, {
  completed: true,
});

// Partial update (only specified fields)
app.tasks.update(db, taskId, {
  description: 'Updated description',
});

// Delete a task
app.tasks.delete(db, taskId);
```

## Create

Returns the ObjectId of the new row:

```typescript
const id = app.tasks.create(db, {
  title: 'New task',
  completed: false,
});
console.log('Created task:', id);
```

## Update

Update one or more fields:

```typescript
app.tasks.update(db, taskId, {
  completed: true,
});

// Update multiple fields
app.tasks.update(db, taskId, {
  title: 'Updated title',
  description: 'Updated description',
  completed: true,
});
```

## Delete

Remove a row by ID:

```typescript
app.tasks.delete(db, taskId);
```

## With React Hooks

Use `useJazz()` to get the database instance:

```tsx title="CreateTask.tsx"
import { useJazz } from '@jazz/react';
import { app } from '../generated/client';

function CreateTaskButton() {
  const db = useJazz();

  const handleCreate = () => {
    const id = app.tasks.create(db, {
      title: 'New task',
      completed: false,
    });
    console.log('Created:', id);
  };

  return <button onClick={handleCreate}>Create Task</button>;
}
```

## Creating Related Data

When creating rows with foreign key references, pass the ObjectId:

```typescript
// Create a project first
const projectId = app.projects.create(db, {
  name: 'My Project',
  color: 'blue',
});

// Create an issue referencing the project
const issueId = app.issues.create(db, {
  title: 'New issue',
  status: 'open',
  priority: 'medium',
  project: projectId,  // Reference the project
});

// Create a label for the issue (many-to-many via join table)
const labelId = app.labels.create(db, {
  name: 'bug',
  color: 'red',
});

app.issuelabels.create(db, {
  issue: issueId,
  label: labelId,
});
```

## Mutations and Subscriptions

When you mutate data, all active subscriptions are automatically notified:

```typescript
// Subscription set up elsewhere
const [tasks, loading] = useAll(app.tasks.where({ completed: false }));

// When you create a task, the subscription updates
app.tasks.create(db, { title: 'New', completed: false });
// tasks array now includes the new task
```
