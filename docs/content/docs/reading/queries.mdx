## Framework hooks

Each framework has a reactive binding that re-renders when query results change. See [Framework Patterns](/docs/reference/framework-patterns#query-subscriptions) for side-by-side examples.

| Framework  | API                            | Notes                       |
| ---------- | ------------------------------ | --------------------------- |
| React/Expo | `useAll(query)`                | Returns `T[] \| undefined`  |
| Vue        | `useAll(query)`                | Returns a `ShallowRef`      |
| Svelte     | `new QuerySubscription(query)` | Exposes reactive `.current` |

### The `undefined` loading state

`useAll` and `QuerySubscription` return `undefined` until the first server response arrives. After that, the value is an array&hairsp;—&hairsp;empty (`[]`) if no rows match, or populated with the requested data.
