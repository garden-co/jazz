## Traditional Migrations vs Jazz Lenses

This section will explain how Jazz read/write-time migration lenses differ from traditional linear migration execution.

## CLI Workflow

1. Update current schema.
2. Run Jazz CLI to generate migration stubs.
3. Edit generated stubs where needed.
4. Rebuild and test.

See [Schemas](/docs/schemas) for generated and edited stub examples.
