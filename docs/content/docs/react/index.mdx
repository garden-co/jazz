Jazz provides a first-class React integration with hooks for querying and subscribing to data.

## Installation

```bash
pnpm add @jazz/react @jazz/client @jazz/schema
```

## Quick Start

1. Define your schema in SQL
2. Generate type-safe client code with `@jazz/schema`
3. Wrap your app with `JazzProvider`
4. Use hooks to query and mutate data

## Available Hooks

| Hook | Purpose |
|------|---------|
| `useAll` | Subscribe to multiple rows matching a query |
| `useOne` | Subscribe to a single row by ID |
| `useMutate` | Get mutation functions for a table |
| `useJazz` | Access the database instance directly |

## Type Safety

All hooks are fully typed based on your schema. The generated types ensure:

- Query results match your table definitions
- Mutations validate against column types
- Related data is typed through the `with()` includes
