Jazz provides React hooks for interacting with your data using the generated type-safe client.

### With Mutations

The third element of the returned tuple provides mutation helpers:

<include>../../../examples/react/src/components/TaskList.tsx#task-list-with-mutations</include>

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `query` | `QueryBuilder` | A query builder from the generated client |

### Returns

| Index | Type | Description |
|-------|------|-------------|
| `[0]` | `T[]` | Query results |
| `[1]` | `boolean` | Loading state |
| `[2]` | `MutateAll` | Object with `create()`, `update()`, and `delete()` methods |

<include>../../../examples/react/src/components/TaskDetail.tsx#task-detail</include>

### With Related Data

<include>../../../examples/react/src/components/TaskDetail.tsx#task-detail-with-includes</include>

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `subscribable` | `SubscribableOneWithDb` | A table descriptor or query builder |
| `id` | `string \| null` | The row's ObjectId |

### Returns

| Index | Type | Description |
|-------|------|-------------|
| `[0]` | `T \| null` | The row or null if not found |
| `[1]` | `boolean` | Loading state |
| `[2]` | `MutateOne` | Object with `update()` and `delete()` methods |

## useMutate

Get typed mutation functions without subscribing to data:

<include>../../../examples/react/src/components/UserProfile.tsx#create-user-button</include>

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `table` | `MutableWithDb` | A table descriptor from the generated client |

### Returns

| Property | Type | Description |
|----------|------|-------------|
| `create` | `(data: T) => ObjectId` | Create a new row |
| `update` | `(id: string, data: Partial<T>) => void` | Update a row |
| `delete` | `(id: string) => void` | Delete a row |
