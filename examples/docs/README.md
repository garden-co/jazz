# Docs-focused examples

This directory contains example apps used as the source of snippets for the docs site.

These examples are intentionally allowed to optimize for documentation clarity and snippet coverage instead of minimal app structure.

## Goals

- Keep docs snippets in runnable, typechecked source files.
- Keep the primary examples under `examples/todo-*` focused on product-style example quality.
- Reuse the same schema model and schema generation workflow used by the main examples.

## Packages

- `todo-client-localfirst-ts`
- `todo-client-localfirst-react`
- `todo-server-ts`
- `todo-server-rs`
