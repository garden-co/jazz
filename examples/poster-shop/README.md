# Poster Shop

A collaborative canvas for designing band logos & gig posters. Built with React + Vite + jazz-tools.

## Run

```bash
pnpm --filter poster-shop dev
```

Optional env vars:

- `VITE_JAZZ_APP_ID`: app identifier used by `createJazzClient`.
- `VITE_JAZZ_SERVER_URL`: Jazz server URL (for collaboration), for example `http://127.0.0.1:1625`.

## Build

```bash
pnpm --filter poster-shop build
```
